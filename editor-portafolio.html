Publicar cambios

Para publicar, el editor sube portafolio.html y portafolio.config.json a GitHub (rama ) y Cloudflare despliega solo en ~1 min.

Pega un token de GitHub con permiso de escritura solo en este repositorio. Se guarda únicamente en esta pestaña (se borra al cerrarla). Cómo crearlo: en el documento docs/editor-portafolio-setup.md.
